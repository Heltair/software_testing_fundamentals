| Tytuł      | Sklep internetowy w MrBuggy - dodawanie smartfonów do koszyka |
| ----------- | ----------- |
| Kroki       |        |
| 1.          | Otwórz aplikację "MrBuggy" zainstalowaną na Twoim telefonie. |
| 2.          | Kliknij ikonę znajdującą się w prawym górnym rogu aplikacji, a następnie kliknij na okrąg z liczbą '3'.|
| 3.          | Zostanie przed Tobą otwarte zadanie ze sklepem internetowym. Sprawdź czy do koszyka można dodać wszystkie 34 dostępne smartfony.|
| Rezultat oczekiwany | Wszystkie 34 smartfony zostały pomyślnie dodane do koszyka. |

| Title      | Online store in MrBuggy - adding smartphones to the shopping cart |
| -----------     | ----------- |
| Steps           |        |
| 1.              | Open the "MrBuggy" application installed on your phone. |
| 2.              | Click on the icon located in the upper right corner of the application. Then, click on a circle with a number '3' in it. |
| 3.              | An online store task will open. Check whether it is possible to add all of 34 available smartphones to the shopping cart.|
| Expected result | All of 34 smartphones were successfully added to the shopping cart. |
